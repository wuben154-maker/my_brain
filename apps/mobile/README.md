# apps/mobile — my_brain iOS / Android App

Expo SDK 52 · React Native 0.76 · shared logic from `@my-brain/core`.

## Features in this app

- **Living Brain Home** — full-screen constellation + voice orb
- **Cold start dialogue** — onboarding into first personal star
- **Realtime voice** — Doubao (Volcengine) dialog with barge-in; provider-swappable
- **Provider settings** — LLM + voice keys stored in Secure Store (not in APK/IPA)
- **Capture inbox** — Android share intent → provisional queue → user ingest gate
- **Brain map, memory review, settings** — local SQLite via expo-sqlite
- **Backup / diagnostics** — encrypted backup hooks, diagnostic export panels

## Scripts

```bash
pnpm --filter @my-brain/mobile start          # Metro
pnpm --filter @my-brain/mobile android        # dev client on device/emulator
pnpm --filter @my-brain/mobile ios            # requires Mac + prebuild
pnpm --filter @my-brain/mobile test
pnpm --filter @my-brain/mobile typecheck
```

## Build & install (others)

Full dual-platform guide: **[`docs/mobile/BUILD.md`](../../docs/mobile/BUILD.md)**

| Platform | Command / path |
|----------|----------------|
| Android release APK | `bash apps/mobile/scripts/build-android-release-apk.sh` |
| iOS (Mac) | `expo prebuild --platform ios` → Xcode |
| iOS (no Mac) | `eas build --platform ios --profile preview` |

## API keys

Release builds **do not** ship with your keys. Each user enters them in **Settings → Provider settings**.

Dev-only: copy repo-root `.env.example` → `.env.local`; Metro seeds secure storage via `app.config.js`.

## Do not

- Commit API keys, `.p12`, or `credentials.json`
- Ship long-lived secrets in `app.json` / `extra`
- Bypass user ingest gate from share/OCR/sync paths
