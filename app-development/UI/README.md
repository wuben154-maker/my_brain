# my_brain Mobile UI High-Fidelity Mockups

这些图用于后续 App 实现审阅，遵循 `specs/mobile-app/assets/ui/DESIGN_SYSTEM.md` 的 v2 风格：

- 390×844 手机画布，深色 Warm Ink 主题
- 温暖伴侣感，不使用 HUD / 霓虹 / dashboard
- 四角星、柔光星座、语音光球、口语化中文
- 三意图作为候选决策模型，不作为永久固定导航
